Aplicacion Full-Stack desarrollada con:

## Frontend
- React con typescript
- React Query y Axios para el fetch de datos
- DaysJS para el manejo de horas y fechas
- Ant design y tailwind css para el maquetado y estilado.

## Backend
- Cuenta con una arquitectura orientada a microservicios. Fue desarrollado con spring boot y spring cloud

### Microservicios 

- **spring-security-jwt** --> Para la autenticacion y autorizacion de usuarios
- **finance-service** --> Para la gestión de cobros, medios de pagos y facturas
- **medpatient-service** --> Para consumir y registrar informacion relacionada con los
  medicos y pacientes
- **medicalconsultationservice** --> Microservicio para el registro de consultas medicas


### Tecnologias

- Spring cloud y Spring boot
- Mysql como motor de bases de datos
- JWT y Spring Security para la autenticacion y autorizacion
- Docker para el despliegue
- Jasper Report para la generacion de facturas
